# Teste de Site - Demonstração Profissional

Este projeto é um site de exemplo simples, desenvolvido com HTML, CSS e JavaScript, para demonstrar a integração e o fluxo de trabalho com o GitHub. Ele inclui uma estrutura básica de página, estilos responsivos e um script para interatividade.

## Funcionalidades

- **Página Inicial:** Seção de boas-vindas com chamada para ação.
- **Sobre:** Informações sobre o projeto e suas características.
- **Serviços:** Lista de serviços fictícios.
- **Contato:** Formulário de contato básico (sem funcionalidade de backend).
- **Responsivo:** Design adaptável para diferentes tamanhos de tela.
- **Integração GitHub:** Configurado para versionamento e automação de CI/CD.

## Estrutura do Projeto

```
teste-de-site/
├── index.html
├── styles.css
└── script.js
```

## Como Visualizar Localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/teste-de-site.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd teste-de-site
   ```
3. Abra o arquivo `index.html` em seu navegador web preferido.
